## 📦  Como baixar o projeto:

```bash
    
    #Clonar o repositório
    $ git clone https://github.com/chagas42/reactMovie2

    #Entrar no diretório
    $ cd reactMovie2

    #Instalar as dependências 
    $ yarn install 

    #Iniciar o projeto
    $ yarn start
```
